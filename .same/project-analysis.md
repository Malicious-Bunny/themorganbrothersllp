# Morgan Dauterive LLP - Immigration Law Firm Website

## Project Overview

This is a Next.js website for **Morgan Dauterive LLP**, an immigration law firm specializing in helping undocumented immigrants secure legal status in the United States.

## Technology Stack

- **Framework**: Next.js 14.2.6 with App Router
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Icons**: Tabler Icons React
- **Theme**: Next Themes (dark/light mode support)
- **Package Manager**: Bun (based on bun.lock file)
- **Additional Tools**: Storybook for component development

## Services Offered

The firm specializes in immigration law for undocumented immigrants with the following services:

### Primary Services (with dedicated pages)
- **DACA (Deferred Action for Childhood Arrivals)** - Applications and renewals
- **Asylum Applications** - For those fleeing persecution
- **Adjustment of Status** - Green card applications/family-based immigration

### Additional Services (mentioned in navigation)
- **U Visa** - For crime victims
- **T Visa** - For trafficking survivors
- **VAWA Protection** - Violence Against Women Act petitions
- **TPS** - Temporary Protected Status
- **Cancellation of Removal** - Deportation defense
